




Create a new HTML file called dom-homework.html.
Add a script tag to the head section of the HTML file to include an external JavaScript file called dom-homework.js.
In the body section of the HTML file, create a div element with an id of "myDiv".
Use JavaScript to add the following elements to the myDiv element:
A heading (h1) element with the text "Welcome to DOM homework".
A paragraph (p) element with the text "This is your first DOM homework assignment".
An unordered list (ul) element with three list items (li) containing any text you want.
Add a class of "highlight" to the heading (h1) element.
Add a background color to the heading with the "highlight" class.
Create a button element below the myDiv element with the text "Add new list item".
Add an event listener to the button that adds a new list item to the ul element each time it is clicked.
Each new list item should have a random color and the text "New List Item X", where X is the number of new list item. 
Use JavaScript to add an event listener to the myDiv element that changes the background color to a random color when the user clicks on the myDiv element.
